DialogPlay Directory

DialogPlay Directory とは

DialogPlay Directory は、大規模言語モデル(LLM)を使って自然言語により様々な API（アプリケーション・プログラミング・インターフェース）にアクセスするためのディレクトリーです。 従来は API を利用するためには、事前に API のインターフェースを確認して、プログラム開発を行う必要がありました。 DialogPlay Directory と大規模言語モデルを使うことで、個々のインターフェースを気にすることなく、様々な API を利用することができるようになりました。 DialogPlay Directory は、オープンソースです。API の追加、更新など、貢献を歓迎します。

DialogPlay Directory を利用する方法

1. DialogPlay から利用する

DialogPlay - AI チャットボットでは、使用する API を選択するだけで、シナリオに必要な API を呼びだすことができます。

2. pytool-directory から利用する

Python で pytool-directory ライブラリーを使用します。LangChain のツールとして pytool-directory を利用すると、エージェントが対話内容に応じて DialogPlay Directory を参照して必要な API を利用します。
